# Chatbot demo (MOVE)

Cette démo est utilisée dans le cadre du projet MOVE. Le chatbot a été adapté à l’usage spécifique de MOVE. Les modifications sont les suivantes :

- Ajout de chat_tools spécifiques au projet :
    - Obtenir l’horaire des activités
    - Obtenir les données d’activité physique

- Intégration de la machine à états dans le nœud

- Prompts de départ adaptés

- Fichier de lancement modifié pour intégrer les nouveaux nœuds

## Utiliser le chatbot

```bash
ros2 launch chatbot_emotion chatbot.launch.xml
```
