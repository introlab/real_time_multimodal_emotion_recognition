# real_time_multimodal_emotion_recognition
This project supports the paper “Real-Time Multimodal Emotion Recognition for an Empathic Socially Assistive Robot: A Pilot Study,” which will be presented at the 18th International Conference on Social Robotics (ICSR + Arts 2026).


## Setup

This project requires the T-Top platform: https://github.com/introlab/t-top

Please refer to the [T-Top Getting Started](https://github.com/introlab/t-top/tree/ros2/documentation/getting_started) page for instructions on setting up this project.
